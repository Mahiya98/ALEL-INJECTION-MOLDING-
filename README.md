# Injection Molding Dashboard

A live, filterable production dashboard for Google Sheets (no Apps Script).

## Setup
1. Open the source Google Sheet → File → Share → **Publish to web** → CSV.
2. Set sharing to "Anyone with the link can view".
3. Edit `script.js` if you ever change `SHEET_ID` or `GID`.
